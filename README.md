# Coming Soon & Maintenance Mode Lite

Show a clean, branded coming-soon or maintenance page to visitors while you
build — admins still see the live site. Part of the
[Zub Plugin Factory](https://github.com/ZubeidHendricks/wp-plugin-factory).

## What it does

One switch hides your site behind a fast, polished splash page. Pick
**Coming Soon** (HTTP 200) or **Maintenance** (HTTP 503 + Retry-After, correct
for SEO during downtime). Logged-in admins bypass it automatically and get a
toolbar warning that it's on. The login page is never blocked.

## Settings

**Settings → Coming Soon** — enable, mode, headline, message, background colour,
and logo (Pro).
